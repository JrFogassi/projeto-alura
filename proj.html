SQUAD

Durante as primeiras semanas do curso houve uma divisão da turma entre equipes, os chamados squads, visando a interação entre colegas e capacitação para atuar em equipe. Cada squad possui um capitão que reúne as metas da semana e ajuda no que for necessário para que as mesmas sejam concluídas.

PROJETOS

Pong

Utilizando as ferramentas fornecidas pelo site Scratch fomos capazes de recriar um dos primeiros jogos de videogames já criados, o Pong. E após isso, fomos desafiados a fazer o Pong utilizando a linguagem de programação JavaScript

FreeWay

Utilizando o site p5.js recriamos outro jogo clássico, FreeWay, onde aplicamos nossos conhecimentos de JavaScript

Barbearia Alura

Após aprendermos sobre HTML5 e CSS3 conseguimos criar um site de uma barbearia, onde montamos uma pagina home, uma de produtos e outra de contatos
