SQUAD

Durante as primeiras semanas do curso houve uma divisão da turma entre equipes, os chamados squads, visando a interação entre colegas e capacitação para atuar em equipe. Cada squad possui um capitão que reúne as metas da semana e ajuda no que for necessário para que as mesmas sejam concluídas.

PROJETOS

A didática do curso de Javascript se baseia na inserção dos estudantes dentro de projetos. Com o objetivo de aprendermos e desenvolvermos a lógica de programação, os conceitos básicos foram passados durante o desenvolvimento de alguns programas, como o cálculo de IMC e o jogo de adivinhação dos números, os quais contam com interações com o usuário. Através dessa didática conseguimos compreender como os sites e programas que utilizam javascript funcionam “por baixo dos panos”.

Pong

Utilizando as ferramentas fornecidas pelo site Scratch fomos capazes de recriar um dos primeiros jogos de videogames já criados, o Pong. Além disso, com o intuito de aprimorar a lógica de programação, fomos desafiados a fazer o jogo utilizando a linguagem de programação JavaScript.

FreeWay

Após tomarmos conhecimento e aprimorarmos as habilidades em JavaScript, utilizamos o site p5.js para recriar outro jogo clássico, FreeWay, o qual possui mais animações, sons e movimentos.

Barbearia Alura

Ao se tratar de HTML5 e CSS3, foi desenvolvido um site para uma barbearia. Durante o desenvolvimento construímos e aprimoramos nossos conhecimentos. O site da Barbearia Alura conta com uma página home, uma de produtos e outra de interação com o usuário.
